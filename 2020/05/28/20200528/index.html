基于Github和Hexo搭建博客

Created2020-05-28Updated2020-05-28

  • #搭建博客的方法

  • 进入D:/Blog/yefeiblog

  • 右键git bash

  • 连接github账号

    • git config –global user.name “yefei521”
    • git config –global user.email “1695618581@qq.com“

  • 新建文章：hexo new post “article title”

    • 在yefeiblog/下有文件 .md，一个存放图片等数据，一个就是文章，在这个文件里编辑文章（markdown语法）

  • hexo g (生成静态网页)

  • hexo s (本地预览效果)

  • hexo d (上传到github)，成功后可以进入https://yefei521.github.io/ 查看

  • 写博客

    • 让博文列表不显示全部内容可以在随意的位置添加一行
